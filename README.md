## MIME

This package provides a comprehensive set of utilities for working with MIME types, including parsing, validation, and detection. It supports a wide range of MIME types and allows for easy manipulation and comparison of MIME type objects.

## Features

- **Parsing**: Convert MIME type strings into structured objects with type, subtype, and parameters.
- **Detection**: Automatically detect MIME types from file content.
- **File Extensions**: A list of common file extensions.
